# MovieAds fresh concept

Статичный визуальный прототип обновленного лендинга MovieAds для монетизации сайтов веб-мастеров видеорекламой.

## Что внутри

- `index.html` - страница концепта.
- `assets/movieads-fresh-concept.css` - стили и адаптив.
- `assets/movieads-logo.png` - текущий логотип MovieAds.

## Идея

Концепт уводит сайт от шаблонного лендинга к более современному platform/dashboard-ощущению: первый экран с оффером, калькулятором дохода, визуальным мокапом рекламного блока, карточками форматов и блоком качества рекламных кампаний.

## Запуск

Откройте `index.html` в браузере или включите GitHub Pages для ветки `main`.
